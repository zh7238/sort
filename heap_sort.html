🌲 堆積排序法演示

操作原理

  1. 先把陣列建成最大堆積（Max Heap），根節點永遠是目前最大值。
  2. 將根節點（最大值）與堆尾交換，把最大值固定在尾端。
  3. 縮小堆積範圍後，對根節點做 heapify 恢復最大堆積。
  4. 重複取最大值到尾端，直到排序完成。

操作過程步驟

  1. 從最後一個非葉節點開始，向前做 heapify，建立最大堆積。
  2. 堆積建立後，array[0] 為當前最大值。
  3. 交換 array[0] 與目前堆尾 array[end]。
  4. end 左移，已排好元素區間逐漸擴大。
  5. 對新根節點做 heapify，恢復最大堆積。
  6. 反覆直到 end = 0。

時間複雜度分析

最壞情況：O(n log n)
建堆 O(n) + 每輪取最大值並 heapify O(log n)
平均情況：O(n log n)
典型資料下也維持穩定的 n log n 效率
最佳情況：O(n log n)
即使資料接近有序，仍需完整堆化與取值流程
空間複雜度：O(1)
原地排序，不需要額外大型暫存陣列
